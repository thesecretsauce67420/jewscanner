# JewScanner
A Free And Open Source Player Tracker For Source Games

# HOW TO SETUP:
Firstly, use pip to install requirements.txt, After you installed it, Create servers.txt and config.json
in config.json do this basic setup:
{
  "TOKEN": "token_here",
  "ALLOWED_CHANNEL_ID": ur_channel_id,
  "GUILD_ID": ur_guild_id,
  "SERVERS_FILE": "servers.txt"
}
Fill out the info needed, then invite the bot you made to the server you want, Put the channel id and guild id of the server in the config, and the bot token in the config.
After this, run python JewScanner.py.
The Bot should go online, try using /addip to add a server ip, then /players or /findplayer name_here

# Required stuff:
Python
A Computer able to run a dsc bot and multiple threads

Thats all!
